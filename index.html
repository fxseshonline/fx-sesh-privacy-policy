4. Data Storage and Security

4.1 Where Your Data is Stored

Cloud Storage (Authenticated Users):

  • Stored in Google Cloud Firestore (servers in us-central1)
  • Journal images stored in Firebase Cloud Storage
  • Data transmitted using industry-standard encryption (TLS/SSL)

Local Storage (Anonymous Users):

  • Data is stored locally on your device
  • Not backed up to the cloud
  • Lost if you uninstall the app or reset your device

4.2 Data Encryption

In Transit: All data transmitted between your device and our servers is encrypted using TLS 1.2 or higher.

At Rest: Data stored in Firebase is encrypted at rest using Google's encryption standards. Journal entries are stored securely in Google Cloud Firestore using industry-standard encryption, but are not end-to-end encrypted.

Administrative Access: Limited authorized personnel may have controlled technical access to backend systems. Your personal data is never accessed except where required by law or explicitly requested for troubleshooting.

4.3 Security Measures

  • Secure authentication protocols
  • Access controls and authorization rules
  • Regular security updates
  • Monitoring for unauthorized access

Note: No method of transmission or storage is 100% secure. We cannot guarantee absolute security of your information.
