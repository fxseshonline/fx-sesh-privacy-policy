## 4. Data Storage and Security

### 4.1 Where Your Data is Stored

**Cloud Storage (Authenticated Users):**
- Your data is stored in Google Cloud Firestore (servers located in us-central1)
- Journal images are stored in Firebase Cloud Storage
- Data is transmitted using industry-standard encryption (TLS/SSL)

**Local Storage (Anonymous Users):**
- Data is stored locally on your device
- Not backed up to the cloud
- Lost if you uninstall the app or reset your device

### 4.2 Data Encryption

**In Transit:**
- All data transmitted between your device and our servers is encrypted using TLS 1.2 or higher

**At Rest:**
- Data stored in Firebase is encrypted at rest using Google's encryption standards
- **Important:** Journal entries are stored securely in Google Cloud Firestore using industry-standard encryption. However, they are not end-to-end encrypted, meaning authorized backend infrastructure may technically access them.

**Administrative Access:**  
As part of operating the App and maintaining secure cloud infrastructure, limited authorized personnel may have controlled technical access to backend systems. However, your personal data is never viewed, accessed, or processed by staff except where required by law or where you explicitly request troubleshooting support.

### 4.3 Security Measures

We implement reasonable security measures including:
- Secure authentication protocols
- Access controls and authorization rules
- Regular security updates
- Monitoring for unauthorized access

**However, no method of transmission or storage is 100% secure.** We cannot guarantee absolute security of your information.
